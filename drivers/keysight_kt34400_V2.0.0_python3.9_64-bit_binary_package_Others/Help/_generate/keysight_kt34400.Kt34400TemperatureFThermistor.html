configure(self: keysight_kt34400.keysight_kt34400.Kt34400TemperatureFThermistor, resolution: float) → None¶

Configures the meter for 4-wire thermistor temperature measurements.

Parameters:

resolution (float) – Measurement resolution. Select from Least (4.5 digits), Default (5.5 digits) and Best (6.5 digits). Higher resolutions (6.5 digits) result in slower measurement speeds.

None
