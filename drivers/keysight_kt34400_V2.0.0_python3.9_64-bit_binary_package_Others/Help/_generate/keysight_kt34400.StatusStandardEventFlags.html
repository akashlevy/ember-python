StatusStandardEventFlags¶

class StatusStandardEventFlags¶

StandardEventFlags enumeration with values corresponding to the 8-bit StandardEvent registers.

Members:

COMMAND_ERROR : Bit 5, value 32, of the Standard Event Register indicating Command Error.

DEVICE_DEPENDENT_ERROR : Bit 3, value 8, of the Standard Event Register indicating Device Dependent Error.

EXECUTION_ERROR : Bit 4, value 16, of the Standard Event Register indicating Execution Error.

OPERATION_COMPLETE : Bit O, value 1, of the Standard Event Register indicating Operation Complete.

POWER_ON : Bit 7, value 128, of the Standard Event Register indicating Power On.

QUERY_ERROR : Bit 2, value 4, of the Standard Event Register indicating Query Error.

REQUEST_CONTROL : Bit 1, value 2, of the Standard Event Register indicating Request Control.

USER_REQUEST : Bit 6, value 64, of the Standard Event Register indicating User Request.

NONE : None, value 0, indicates no register bits are set.
