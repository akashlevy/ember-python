  2. configure(self: keysight_kt34400.keysight_kt34400.Kt34400SecondaryDisplayDCVoltage, range: float, resolution: keysight_kt34400.keysight_kt34400.Resolution) -> None

    Resets all (Secondary) measurement parameters and then configures the DMM for DC voltage measurements. This method does not initiate a scan.

    Parameters:

    • range (float) - The expected signal value. Positive values only.

    • resolution (Resolution) - Measurement resolution. Select from Least (4.5 digits), Default (5.5 digits) and Best (5.5 digits). Higher resolutions (5.5 digits) result in slower measurement speeds.

    Returns:

    None

    remarks:
  3. This method resets all frequency measurement parameters and trigger parameters to their default values. Then, it configures the meter for frequency measurements but does not initiate the measurement.
  4. Desired range is 100 mV(MIN), 1 V, 10 V, 100 V, 1000 V(MAX).
  5. Instrument can automatically select the measurement range using Kt34400.range.
  6. If input value is greater than that can be measured on the selected range, "9.9E+37" is returned.
