    1. read_erase(self: keysight_kt34400.keysight_kt34400.Kt34400Measurement, max_readings: int) -> numpy.ndarray[numpy.float64]

      Reads and erases measurements from reading memory up to the specified maxReadings.

      Parameters:

      • max_readings (int) - Specific number of readings to be read and erased from memory.

      Returns:

      The removed values.

      Return type:

      numpy.ndarray<with dtype:double>

      remarks:
    2. The measurements are erased from memory starting with the oldest measurement first.
    3. It does not wait for all readings to complete. It sends the readings that are complete at the time the instrument receives the command.
    4. No error is generated if the reading memory contains less readings than requested. In this case, all available readings in memory are read and deleted.
    5. The instrument clears all measurements from reading memory when the measurement configuration changes.

    • remove_readings(*args, **kwargs)¶

    Overloaded function.

    1. remove_readings(self: keysight_kt34400.keysight_kt34400.Kt34400Measurement, num_readings: int) -> numpy.ndarray[numpy.float64]

      Reads and erases the specified number of readings from memory.

      Parameters:

      • num_readings (int) - Specific number of readings to be read and erased from memory.

      Returns:

      The removed values.

      Return type:

      numpy.ndarray<with dtype:double>

      remarks:
    2. The readings are erased from memory starting with the oldest reading first. The purpose of this method is to allow you to periodically remove readings from memory during a series of measurements to avoid a reading memory overflow. If the specified number of readings are not yet present in memory when this routine is called, an instrument error will result and the data fetch will fail.

    3. remove_readings(self: keysight_kt34400.keysight_kt34400.Kt34400Measurement, num_readings: int, wait: keysight_kt34400.keysight_kt34400.DataRemove) -> numpy.ndarray[numpy.float64]

      Reads and erases the specified number of readings from memory.

      Parameters:

      • num_readings (int) - Specific number of readings to be read and erased from memory.

      • wait (DataRemove) - The WAIT parameter is specified, in which case the query waits until numReadings measurements are available.

      Returns:

      The removed values.

      Return type:

      numpy.ndarray<with dtype:double>

      remarks:
    4. The readings are erased from memory starting with the oldest reading first. The purpose of this method is to allow you to periodically remove readings from memory during a series of measurements to avoid a reading memory overflow. If the specified number of readings are not yet present in memory when this routine is called, the query waits until numReadings measurements are available.
